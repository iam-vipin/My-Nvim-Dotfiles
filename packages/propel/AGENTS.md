# Propel

New Storybook component library (`@plane/propel`). Actively developed replacement for `@plane/ui`.

Run `pnpm --filter=@plane/propel storybook` to start Storybook on port 6006.

See [docs/TYPESCRIPT.md](../../docs/TYPESCRIPT.md) for conventions and [docs/DESIGN_SYSTEM.md](../../docs/DESIGN_SYSTEM.md) for styling.
